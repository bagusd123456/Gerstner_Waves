# Gerstner Waves
Unity URP Water Shader with Buoyancy

Package Requirements :
- Unity New Input System
- Universal Render Pipeline

Controls :
- WASD to move.
- Mouse scroll to adjust speed.

Video : https://www.youtube.com/watch?v=fNY3LOfjG5c
